# Demo 3D Rendering Engine
*A lightweight Linux renderer aiming to be a sandbox environment for graphics and physics*

#### Engine goals
- To provide a platform to develop cross-platform desktop applications
- To support a physics library such as [Bullet](https://www.example.com](https://github.com/bulletphysics/bullet3)) or [PhysX](https://nvidia-omniverse.github.io/PhysX/physx/5.5.0/)

#### Personal goals
- Improve my understanding od 3D graphics
- Implement an animation system for character animation
- To explore a modern graphics framework and take advantage of modern hardware

#### Features
- Basic pipeline setup using uniform buffers
- Sdl2 window and input system
- Gltf format loading (currently limited to 1 texture)
